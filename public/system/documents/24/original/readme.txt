Tahometer Agent 2.0.5

Installation
------------

If you have problems with running Tahometer Agent under Linux 64-bit install the required packages using the following command: "sudo apt-get install lib32stdc++6 ia32-libs libc6-i386 lib32gcc1".

Установка
---------

При возникновении проблем с запуском Tahometer Agent под Linux 64-bit установите необходимые пакеты с помощью комманды: "sudo apt-get install lib32stdc++6 ia32-libs libc6-i386 lib32gcc1".
